## Quick Population Script

If you want to quickly populate cars via WordPress CLI (if available):

```bash
# Add cars via WordPress CLI
wp post create --post_type=car --post_title="2023 BMW X5" --post_content="Description here"

# Assign categories
wp term create car_category "SUV"
wp post term add POST_ID car_category "SUV"
```
